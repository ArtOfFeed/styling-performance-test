# CSS in JS Performance comparison

## Setup 
Setup: 

```bash
git clone https://github.com/ArtOfFeed/styling-performance-test.git
cd styling-performance-test
yarn
```

## Run

Terminal 1: 

```
yarn start
```

Terminal 2: 

```
npx ts-node --compiler-options '{ "module" : "commonjs" }' ./src/scripts/test.tsx
```
